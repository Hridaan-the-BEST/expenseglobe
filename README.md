#💰 SpendSmart - An Expense Tracker
A simple and beautiful personal expense tracker built with HTML, CSS & JavaScript.

SpendSmart's

✨ Features
-Add Income & Expenses
-Real-time Balance Calculation
-Categorize transactions (Food, Transport, Shopping, etc.)
-Delete transactions
-Data saved in browser (no backend needed)
-Fully Responsive Design

# 🚀 Live Demo:
👉 View Live Website(First Version) - https://hridaan-the-best.github.io/SpendSmart

🛠️ Tech Stack
-HTML5
-CSS3
-Vanilla JavaScript

📱 How to Use
Enter description, amount, type (Income/Expense) and category
Click "Add Transaction"
Your balance updates automatically
All data is saved in your browser

🏠 How to Run Locally
-Download or clone the project
-Open index.html in any browser
-Start tracking your expenses!
