# 💰 SpendSmart - Expense Tracker

A simple and beautiful personal expense tracker built with HTML, CSS & JavaScript.

![SpendSmart](https://via.placeholder.com/800x200/667eea/ffffff?text=SpendSmart)

---

## ✨ Features

- Add Income & Expenses
- Real-time Balance Calculation
- Categorize transactions (Food, Transport, Shopping, etc.)
- Delete transactions
- Data saved in browser (no backend needed)
- Fully Responsive Design

---

## 🚀 Live Demo

👉 [View Live Website](https://hridaan-the-best.github.io/SpendSmart)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage

---

## 📱 How to Use

1. Enter description, amount, type (Income/Expense) and category
2. Click **"Add Transaction"**
3. Your balance updates automatically
4. All data is saved in your browser

---

## 🏠 How to Run Locally

1. Download or clone the project
2. Open `index.html` in any browser
3. Start tracking your expenses!
